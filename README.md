# Intruder-Detection-System
Developed an Intelligent Intruder Detection System that bridges computer vision with automated network communications. Built with Python, OpenCV, and YOLOv8, the system monitors video streams for perimeter breaches within localized Regions of Interest (ROI). Upon threat detection, a multi-threaded architecture dispatches cryptographic email alerts with frame snapshots via SMTP. It concurrently executes a background IMAP parser to listen for user confirmation ('yes'/'no'), implementing a human-in-the-loop verification protocol that dynamically scales to trigger secondary emergency notifications or stand down based on real-time feedback.
